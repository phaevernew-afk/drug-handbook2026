使用說明
1
點「載入目前資料」讀取網站現有的 drugs.json，或直接上傳 Excel（.xls / .xlsx）建立新資料表。
2
在下方表格中直接點選儲存格編輯、新增列、或刪除列來修改 / 新增藥品。
3
完成後按「下載 drugs.json」，將下載的檔案覆蓋網站 GitHub repo 中的 drugs.json 並上傳（commit）即可更新查詢頁。
